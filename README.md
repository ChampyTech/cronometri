# ⏳ Cronometri

Un'unica applicazione web per calcolare dinamicamente il tempo vissuto (in anni, mesi, giorni e ore) di diverse persone.  
Per visualizzare il cronometro di una persona specifica, basta inserire il suo codice identificativo alla fine dell'URL principale:

```text
https://champytech.github.io/cronometri/[IDENTIFICATIVO]
```

**Esempi**:

* **Profilo DC:** [https://champytech.github.io/cronometri/dc](https://champytech.github.io/cronometri/dc)
* **Profilo BM:** [https://champytech.github.io/cronometri/bm](https://champytech.github.io/cronometri/bm)

*Nota: Se visiti l'URL principale senza specificare un nome, il sistema mostrerà il profilo di **DC** come predefinito.*

**Persone disponibili**:

Puoi visualizzare l'elenco completo delle persone disponibili direttamente nelle prime righe di [questo file](https://champytech.github.io/cronometri/script.js).
